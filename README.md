# TruthLens AI

TruthLens AI is a web-based fake news detection system that uses Machine Learning and Natural Language Processing to classify news as REAL or FAKE.

## Features

- Fake news detection
- Confidence percentage
- AI-style explanation
- Modern web interface
- Flask-based web application
- Machine Learning model using TF-IDF and Passive Aggressive Classifier

## Dataset

Dataset used:
https://www.kaggle.com/datasets/bhavikjikadara/fake-news-detection

The dataset is not uploaded due to GitHub file size limits.

## Technologies Used

- Flask
- Scikit-learn
